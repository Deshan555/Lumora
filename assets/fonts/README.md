# Font Files Placeholder

The fonts will be loaded via google_fonts package at runtime.
No need to download TTF files manually.

If you want to bundle fonts:
1. Download Poppins from: https://fonts.google.com/specimen/Poppins
2. Download DM Sans from: https://fonts.google.com/specimen/DM+Sans
3. Place TTF files in this directory
4. Update pubspec.yaml fonts section

For now, using google_fonts package which downloads fonts on demand.
